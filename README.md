# Jason Binong Portfolio

Personal portfolio for Jason Binong, focused on AI, data, information systems, and student technology projects.

## Live Projects Featured

- Nexus AI
- CareerLens
- LearnWise
- 15 Weeks at UMBC

## How To Run

Open `index.html` in a browser.

## Deploy With GitHub Pages

1. Create a GitHub repository named `Jason-Binong-Portfolio` or `jasonbinong.github.io`.
2. Upload these files to the repository.
3. Go to **Settings** -> **Pages**.
4. Choose **Deploy from a branch**.
5. Select the `main` branch and `/root`.
6. Save.

If the repository is named `jasonbinong.github.io`, the portfolio will publish at:

```text
https://jasonbinong.github.io/
```

If the repository is named `Jason-Binong-Portfolio`, it will publish at:

```text
https://jasonbinong.github.io/Jason-Binong-Portfolio/
```
